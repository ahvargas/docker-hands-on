####  Consultando los contenedores

Las contenedores se pueden consultar con el siguiente comando :

```bash
» docker ps
```

```bash
CONTAINER ID        IMAGE               COMMAND             CREATED             STATUS              PORTS               NAMES
```

- Por defecto solo muestra los contenedores en ejecución. <!-- .element: class="fragment"  -->
- Es por eso que no se muestra ningún contenedor. <!-- .element: class="fragment"  -->
